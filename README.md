# CampusTalk - 대학생 커뮤니티 웹사이트

에브리타임과 같은 대학생 커뮤니티 플랫폼입니다. 게시판, 중고거래, 시간표 등 대학생활에 필요한 모든 기능을 제공합니다.

## ✨ 주요 기능

- **커뮤니티 중심 디자인**: 첫 화면에서 바로 게시판 확인 가능
- **반응형 디자인**: 모든 기기에서 완벽하게 작동
- **현대적인 UI/UX**: 아름다운 그라디언트와 애니메이션
- **부드러운 스크롤**: 섹션 간 자연스러운 이동
- **모바일 최적화**: 햄버거 메뉴와 터치 제스처 지원
- **인터랙티브 요소**: 호버 효과와 스크롤 애니메이션
- **접근성**: 키보드 네비게이션 지원

## 🚀 시작하기

1. 프로젝트 파일들을 다운로드하거나 클론합니다
2. `index.html` 파일을 웹 브라우저에서 엽니다
3. 또는 로컬 서버를 실행합니다:

   ```bash
   # Python 3
   python -m http.server 8000

   # Node.js
   npx serve .
   ```

## 📁 파일 구조

```
├── index.html          # 메인 HTML 파일
├── styles.css          # CSS 스타일시트
├── script.js           # JavaScript 기능
├── login.html          # 로그인/회원가입 페이지
├── login-styles.css    # 로그인 페이지 스타일
├── login-script.js     # 로그인 페이지 기능
├── free-board.html     # 자유게시판 전용 페이지
├── class-info.html     # 수업정보 게시판 페이지
├── exam-info.html      # 시험정보 게시판 페이지
├── job-info.html       # 취업정보 게시판 페이지
├── board-styles.css    # 게시판 페이지 스타일
├── board-script.js     # 게시판 페이지 기능
├── write-post.html     # 게시글 작성 페이지
├── write-post-styles.css # 글쓰기 페이지 스타일
├── write-post-script.js  # 글쓰기 페이지 기능
└── README.md           # 프로젝트 설명서
```

## 🎨 섹션 구성

1. **Hero Section**: 간결한 브랜드 소개와 가입/둘러보기 버튼
2. **Board Section**: 🔥 실시간 인기 게시판 (자유게시판, 수업정보, 시험정보, 취업정보)
3. **Market Section**: 중고거래 (전공책, 전자기기, 의류, 기타)
4. **Timetable Section**: 시간표 관리 및 공유

## 🛠️ 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: Flexbox, Grid, 애니메이션
- **JavaScript (ES6+)**: 인터랙티브 기능
- **Google Fonts**: Noto Sans KR 폰트

## 📱 반응형 브레이크포인트

- **Desktop**: 1200px 이상
- **Tablet**: 768px - 1199px
- **Mobile**: 767px 이하

## 🎯 주요 기능 설명

### 게시판 시스템

- **독립적인 게시판 페이지**: 각 카테고리별로 전용 페이지 제공
  - 💬 **자유게시판** (`free-board.html`): 학생들의 자유로운 소통 공간
  - 📚 **수업정보** (`class-info.html`): 수업 관련 공지사항 및 정보
  - 📝 **시험정보** (`exam-info.html`): 시험 일정 및 관련 정보
  - 💼 **취업정보** (`job-info.html`): 채용 정보 및 취업 관련 팁
- **전용 글쓰기 페이지** (`write-post.html`): 게시글 작성 전용 인터페이스
  - **게시판 선택**: 작성할 게시판을 선택하여 게시
  - **파일 첨부**: 최대 5개 파일 첨부 지원 (이미지, PDF, 문서)
  - **미리보기**: 게시 전 내용 미리보기 기능
  - **게시글 옵션**: 익명 작성, 댓글 허용, 알림 설정
- **고급 기능**: 검색, 정렬, 페이지네이션 지원
- **실시간 상호작용**: 좋아요, 댓글, 조회수 표시
- **반응형 디자인**: 모든 기기에서 최적화된 게시판 경험

### 중고거래 플랫폼

- 카테고리별 상품 분류 (전공책, 전자기기, 의류, 기타)
- 상품 이미지 및 상세 정보
- 위치 기반 거래 정보
- 판매하기 및 더보기 기능

### 시간표 관리

- 주간 시간표 표시
- 과목별 색상 구분
- 과목 추가 및 공유 기능
- 학기별 시간표 관리

### 사용자 인증

- 로그인/회원가입 모달
- 비밀번호 찾기 기능
- 로그인 필요 서비스 안내

## 🔧 커스터마이징

### 색상 변경

`styles.css`에서 CSS 변수를 수정하여 테마 색상을 변경할 수 있습니다:

```css
:root {
  --primary-color: #6366f1;
  --secondary-color: #764ba2;
  --text-color: #333;
  --background-color: #f8fafc;
}
```

### 콘텐츠 수정

`index.html`에서 텍스트와 이미지를 원하는 내용으로 변경할 수 있습니다.

### 기능 추가

`script.js`에 새로운 JavaScript 기능을 추가할 수 있습니다.
